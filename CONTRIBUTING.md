# Contributing Guide

These guides are built through collaboration with the global commodity trading community. Your contributions are invaluable! 🙏

## 📋 Types of Contributions

### 1. 🐛 Bug Reports
If you find incorrect, incomplete, or outdated information in the guides:

- Open a **GitHub Issue**
- Clearly explain the problem
- Provide sources or references
- Suggest corrections

### 2. 💡 New Guide Suggestions
If you think a new guide on a topic is needed:

```
Title: [Guide Suggestion] Your Topic
Description: Why is this guide needed?
Target Audience: Beginners / Advanced / Professionals
Estimated Length: Short (< 500 words) / Medium (500-1500) / Long (1500+)
```

### 3. ✏️ Content Corrections
- Fix spelling and grammar errors
- Clarify ambiguous explanations
- Update outdated information
- Improve sources

### 4. 📖 Translations
Help translate guides into other languages. Supported languages:
- English (Current)
- Turkish (Current)
- Arabic (Planned)
- Mandarin (Planned)

### 5. 📊 Data and Graphics
- Add market charts
- Create infographics
- Prepare price comparison tables
- Draw flow diagrams

## 🚀 Getting Started

### 1. Fork the Repository
```bash
git clone https://github.com/skyra-cip/commodity-trading-guides.git
cd commodity-trading-guides
```

### 2. Create a New Branch
```bash
git checkout -b feature/new-guide-name
# or
git checkout -b fix/error-fix
```

### 3. Make Your Changes
- Write guides to the wiki folder
- Use `.md` (Markdown) format
- Follow title structure for SEO and readability

### 4. Commit and Push
```bash
git add .
git commit -m "Descriptive commit message"
git push origin feature/new-guide-name
```

### 5. Open a Pull Request
Create a Pull Request on GitHub:
- **Title:** Summarize the change
- **Description:** Explain why this change is needed
- **References:** Add reference links

## 📝 Content Writing Standards

### Title Structure
```markdown
# Main Topic (H1)
## Subtitle (H2)
### Detail Heading (H3)
```

### Language
- **Professional:** Use technical terms appropriately
- **Clear:** Explain complex concepts
- **Concise:** Keep sentences short (< 20 words)
- **Explicit:** Provide examples and explanations

### Examples
- Show product specifications in tables
- Explain trading procedures step-by-step
- Include real-world examples

### Sources
Support each claim with credible sources:
- **Standards:** ASTM, ISO, SWIFT
- **Organizations:** ICC, ITC, WTO
- **Industry:** Platts, Argus, LME
- **Academic:** Research papers, reports
